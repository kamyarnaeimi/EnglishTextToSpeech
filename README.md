# English Text To Speech (Google Colab)

This project converts English text into speech using gTTS and plays the audio directly inside Google Colab.

## Features
- Convert English text to MP3
- Play audio directly in Colab
- Beginner-friendly
- Uses Google Text-to-Speech (gTTS)

## How to Run
1. Open Google Colab
2. Upload the Python file
3. Run the code
4. Enter an English sentence

## Requirements
pip install gtts
